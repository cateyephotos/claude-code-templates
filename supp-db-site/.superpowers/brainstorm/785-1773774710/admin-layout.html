Where should Email Sequences live in the admin?

Your admin dashboard already has a sidebar with navigation. Pick where the email system fits.

A

New Sidebar Section

Add an "Email Campaigns" item to the existing admin sidebar, alongside your current dashboard sections. Clicking it reveals sub-pages: Sequences, Templates, Analytics. Everything stays in one admin shell.

B

Dedicated Email Tab

Add a top-level tab/icon in the admin header bar. Email management gets its own full-width workspace separate from the analytics dashboard. Like switching between "Analytics" and "Email" modes.

C

Expandable Sidebar Group

Add a collapsible "Email" group in the sidebar that expands to show: Sequences, Subscribers, Analytics. Keeps the sidebar organized as you add more admin features over time. Most scalable.
