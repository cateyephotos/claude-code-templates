# Evidence Evaluation: Bacopa monnieri (ID 1) - Mode 2 Evidence Update

## Evaluation Metadata
- **Date**: 2026-03-05
- **Papers Evaluated**: 3
- **Framework**: GRADE-informed assessment

---

## Paper 1: Tiemtad et al. 2026

| Criterion | Assessment |
|-----------|------------|
| **GRADE** | HIGH quality |
| **Evidence Level** | Level 1 (SR+NMA of RCTs) |
| **Risk of Bias** | Low -- comprehensive search across MEDLINE, Cochrane, Scopus; NMA methodology with SUCRA rankings |
| **Precision** | Excellent -- 29 RCTs, n=2107, significant p-values |
| **Consistency** | High -- consistent direction for working memory (SMD=2.03, 95% CI: 1.28-2.78, p<0.05) |
| **Directness** | High -- directly compares Bacopa vs Ginkgo vs placebo for cognitive function |
| **Quality Score** | 9/10 |

### Key Findings
- High-dose Brahmi (>=600mg/day) significantly improved working memory vs placebo (SUCRA 100%).
- Also superior vs low-dose Brahmi, high-dose Ginkgo, low-dose Ginkgo.
- Short-term memory also significant.
- No significant differences in attention or processing speed.

## Paper 3: Lopresti & Smith 2025

| Criterion | Assessment |
|-----------|------------|
| **GRADE** | MODERATE quality |
| **Evidence Level** | Level 2 (well-designed RCT) |
| **Risk of Bias** | Low -- double-blind, placebo-controlled, randomized, adequate sample |
| **Precision** | Good -- n=101, adequate for detecting medium effects |
| **Consistency** | MIXED -- negative primary cognitive outcomes contradict some earlier RCTs; positive stress/fatigue align with adaptogenic use |
| **Directness** | High -- directly studies Bacopa extract (Bacumen) in healthy adults |
| **Quality Score** | 7/10 |

### Key Findings
- Negative primary cognitive outcomes at 300mg/day.
- Positive stress reactivity (p=0.03).
- Positive anti-fatigue effects.

### Implications
- Consistent with Tiemtad dose-response finding -- 300mg may be subtherapeutic for cognitive benefits while adequate for stress.
