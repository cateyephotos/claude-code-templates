## Search Limitations

1. **No keyword/text search in bioRxiv tool** — bioRxiv MCP only supports date/category filtering, not keyword search. Preprints could not be systematically searched.
2. **Consensus limited filtering** — Consensus does not support strict date range or study type filters, limiting precision.
3. **No Cochrane Library search** — Cochrane was not searched as a separate database. Any Cochrane reviews indexed in PubMed would have been captured.
4. **No EMBASE/Scopus** — Search limited to PubMed and Consensus. Some non-PubMed-indexed journals may have been missed.
5. **No grey literature search** — Conference abstracts, clinical trial registries (ClinicalTrials.gov), and dissertations were not searched.
